Install

Load as an unpacked extension in Chrome or any Chromium browser.

STEP 01

Clone & Build

git clone github.com/m4cd4r4/recstitch
then npm install && npm run build

STEP 02

Load in Chrome

Go to chrome://extensions, enable Developer Mode, click Load Unpacked, select the dist/ folder.

STEP 03

Record

Click the RecStitch icon, hit Record, use the site normally, then Stop and export.
